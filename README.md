# Notes On Band
Custom Notes on Microsoft Band 1 and 2. Built for Windows 10 for Mobile.

## Implementation details:
- Using C#, Windows 10 SDK 10586, Microsoft Band SDK v1.3

## Main Feature:
- Add custom notes onto Microsoft Band
